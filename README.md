## WhoGot

An app to let others know what's in your fridge and available for sharing

## Technologies used

- Dotenv: 8.2.0
- Express: 4.17.1
- Express-handlebars: 5.2.0
- Express-session: 1.17.1
- Handlebars: 4.7.6
- Mysql: 2.2.5
- Nodemon: 2.0.20
- Sequelize: 6.3.5
- Email-Validator: 2.0.4
